# Application de gestion de tâches (React + Vite)

Application web simple de gestion de tâches développée avec **React** et **Vite**.

Fonctionnalités :

- **Ajouter des tâches** via un champ de saisie et un bouton.
- **Marquer les tâches comme terminées** (case à cocher).
- **Supprimer des tâches** individuellement.
- **Filtrer** les tâches : toutes, en cours, terminées.

## Installation

Assure-toi d'avoir **Node.js** et **npm** installés sur ta machine.

Dans le dossier du projet (`c:\\Documents\\Essai App`) :

```bash
npm install
```

## Lancer l'application en développement

Toujours dans le dossier du projet :

```bash
npm run dev
```

Puis ouvre l'URL affichée dans le terminal (par défaut `http://localhost:5173`).

## Build de production (optionnel)

```bash
npm run build
npm run preview
```
